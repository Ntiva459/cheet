Laplace Transform

Chapter 2
f(t) → F(p)
June 2026
Core Properties
Properties — Formula Sheet Supplement
linearity a·f + b·g → a·F(p) + b·G(p)
1st shift e^(at)·f(t) → F(p − a)
t-mult t · f(t) → − F'(p)
deriv f' f'(t) → p·F(p) − f(0)
deriv f'' f''(t) → p²·F(p) − p·f(0) − f'(0)
integral ∫₀ᵗ f(u) du → F(p) / p
convolution (f * g)(t) → F(p) · G(p)
Workflows
Forward Transform — Inside Out
1
Identify the innermost function (sin, cos, t^n)
2
Apply L[t·f(t)] = −F'(p) if there's a t factor
3
Apply shift F(p−a) last for the e^(at) factor
Example: t · cos(2t) · e^(−t)
① L[cos(2t)] = p/(p²+4)
② −d/dp[p/(p²+4)] = (p²−4)/(p²+4)²
③ replace p → (p+1)
= ((p+1)²−4) / ((p+1)²+4)²
Inverse Transform — Partial Fractions
1
Factor the denominator completely
2
Decompose: linear → A/(p−a), irreducible quadratic → (Bp+C)/(p²+w²)
3
Find constants by substituting roots (plug p = root for each linear factor)
4
For quadratic: match coefficients of p², p¹, p⁰
5
Invert each term using the table
ODE Solving — Cauchy Method
1
Apply L to every term in the ODE
2
Substitute initial conditions x(0), x'(0) immediately
3
Solve algebraically for X(p)
4
Partial fractions on X(p)
5
Invert to get x(t)
⚠ Never forget Initial conditions are substituted at Step 2, not at the end. Missing them costs the whole problem.
Variable Coefficient ODEs
When ODE has t · x'', t · x' terms
1
Transform normally first: L[x''] = p²X(p) − p·x(0) − x'(0)
2
Then apply t-multiplication: L[t·x''] = −d/dp[p²X(p) − p·x(0) − x'(0)]
3
This produces a first-order ODE in p with X'(p) — solve it as a separable ODE
4
Use lim(p→∞) X(p) = 0 as the boundary condition to find the constant C
5
Invert X(p) using the table
⚠ Key difference You end up with X'(p) in the equation — it's no longer pure algebra. Separate variables: dX/X = (...)dp, integrate both sides, then exponentiate.
t-Multiplication Rules — Applied to ODE Terms
t·x(t) L[t·x] → −X'(p)
t·x'(t) L[t·x'] → −d/dp[pX(p) − x(0)]
t·x''(t) L[t·x''] → −d/dp[p²X − px(0) − x'(0)]
After differentiating and substituting ICs
L[t·x''] = −(2p·X(p) + p²·X'(p)) + x(0)
L[t·x'] = −(X(p) + p·X'(p)) + 0
→ collect X'(p) terms → separable ODE
Partial Fractions — Case Reference
Decomposition Forms
Factor in denom Form to use Find constants by
(p − a) A / (p − a) plug p = a
(p − a)² A/(p−a) + B/(p−a)² plug p = a, match coefficients
(p² + w²) (Bp + C) / (p² + w²) match p², p¹, p⁰ coefficients
(p−a)(p²+w²) A/(p−a) + (Bp+C)/(p²+w²) plug p=a for A, then match
Completing the Square — Inverse Trick
When denominator won't factor → complete the square, then match to shifted sin/cos.
Pattern (p−a)² + w²
→ sine w/[(p−a)²+w²] → e^(at)·sin(wt)
→ cosine (p−a)/[(p−a)²+w²] → e^(at)·cos(wt)
Example: 3/(p²−4p+13)
p²−4p+13 = (p−2)²+ 9
w = 3, a = 2
→ e^(2t) · sin(3t)
Applications
Integral Equations — Convolution Method
1
Recognize ∫₀ᵗ x(u)·g(t−u) du as convolution x * g
2
Transform: L[x*g] = X(p)·G(p)
3
Solve algebraically for X(p), then partial fractions, then invert
x(t) + ∫₀ᵗ x(u)·sin(t−u) du = t
X(p) + X(p)·1/(p²+1) = 1/p²
X(p)·(p²+2)/(p²+1) = 1/p²
X(p) = (p²+1) / [p²·(p²+2)]
→ x(t) = ½ + ½·cos(√2·t)
Improper Integrals — f(t)/t Theorem
∫₀^∞ f(t)/t dt = ∫₀^∞ F(p) dp
1
Identify f(t) — the numerator only, ignore the /t
2
Find F(p) = L[f(t)]
3
Integrate F(p) from 0 to ∞ and evaluate limits
Classic: ∫₀^∞ sin(t)/t dt
F(p) = 1/(p²+1)
∫₀^∞ 1/(p²+1) dp = arctan(p)|₀^∞
= π/2 − 0 = π/2
Shifted: ∫₀^∞ e^(−t)·sin(t)/t dt
f(t) = e^(−t)·sin(t)
F(p) = 1/((p+1)²+1) — shift by a=1
∫₀^∞ F(p) dp = arctan(p+1)|₀^∞
= π/2 − arctan(1) = π/2 − π/4 = π/4
⚠ Common mistake arctan(∞) = π/2, NOT 1. With shift, lower limit is arctan(a), not 0.
