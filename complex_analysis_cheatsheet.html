Complex Analysis

Special Math (S) · Exam Cheatsheet · UTCN 2026

Complex Logarithm

Definition (multivalued)
\(\text{Log}(z) = \ln|z| + i(\arg z + 2k\pi), \quad k \in \mathbb{Z}\)
Principal value (k=0)
\(\log z = \ln|z| + i \cdot \text{Arg}(z), \quad \text{Arg}(z) \in (-\pi, \pi]\)
Power function
\(z^a = e^{a \cdot \text{Log}(z)}\)
Key arguments: Arg(positive real) = 0 · Arg(negative real) = π · Arg(+i·r) = π/2 · Arg(−i·r) = −π/2

Elementary Functions

Exponential
\(e^z = e^x(\cos y + i \sin y), \quad z = x+iy\)
Trig (exponential form)
\(\sin z = \dfrac{e^{iz}-e^{-iz}}{2i} \qquad \cos z = \dfrac{e^{iz}+e^{-iz}}{2}\)
Hyperbolic
\(\sinh z = \dfrac{e^{z}-e^{-z}}{2} \qquad \cosh z = \dfrac{e^{z}+e^{-z}}{2}\)
Euler's formula
\(e^{i\theta} = \cos\theta + i\sin\theta\)

Solving Complex Trig Equations

sin(z) = a → substitute w = e^iz
  1. Write exponential form: \(\dfrac{w - 1/w}{2i} = a\)
  2. Multiply by \(2iw\): \(w^2 - 2iaw - 1 = 0\)
  3. Solve quadratic: \(w = ia \pm \sqrt{1 - a^2}\) (check discriminant carefully)
  4. Take complex log: \(iz = \text{Log}(w)\), then divide by \(i\)
  5. Dividing by \(i\): multiply by \(-i\) (since \(1/i = -i\))
cos(z) = a → same but
\(\dfrac{w + 1/w}{2} = a \Rightarrow w^2 - 2aw + 1 = 0\)
sinh(z) = a → substitute w = e^z
\(w^2 - 2aw - 1 = 0\), then \(z = \text{Log}(w)\)

Singularity Classification

Always expand the numerator first — a zero in the numerator reduces the pole order!
TypePrincipal PartHow to spot
Removable 0 terms Limit exists and is finite
Pole order k k terms Finite negative powers, ends at \((z-z_0)^{-k}\)
Essential ∞ terms \(e^{1/z}\), \(\sin(1/z)\) type
Check order by Laurent expansion
\(\frac{\sin z}{z^4} \Rightarrow \frac{z - z^3/6 + \cdots}{z^4} = z^{-3} - \frac{1}{6}z^{-1} + \cdots\)
Lowest power is z⁻³ → pole of order 3, NOT 4

Residue Calculation

Simple pole (order 1)
\(\text{Res}[f, z_0] = \lim_{z \to z_0}(z-z_0)f(z)\)
Pole of order n
\(\text{Res}[f, z_0] = \dfrac{1}{(n-1)!}\lim_{z \to z_0}\dfrac{d^{n-1}}{dz^{n-1}}\left[(z-z_0)^n f(z)\right]\)
Ratio rule — f(z) = g(z)/h(z), z₀ simple zero of h
\(\text{Res}[f, z_0] = \dfrac{g(z_0)}{h'(z_0)}\)
Only works when: pole is order 1 AND g(z₀) ≠ 0
Essential singularity
Must expand Laurent series — residue = coefficient of \((z-z_0)^{-1}\)

Residue Theorem

General formula
\(\oint_C f(z)\,dz = 2\pi i \sum_{k} \text{Res}[f, z_k]\)
Sum over ALL singularities inside C only
Single singularity
\(\oint_C f(z)\,dz = 2\pi i \cdot \text{Res}[f, z_0]\)
Step 1: Find singularities  |  Step 2: Check if inside contour  |  Step 3: Calculate residues  |  Step 4: Sum × 2πi

Cauchy's Integral Formula

General (n-th derivative)
\(\oint_C \dfrac{f(z)}{(z-z_0)^n}\,dz = \dfrac{2\pi i}{(n-1)!}\,f^{(n-1)}(z_0)\)
n = 1
\(2\pi i \cdot f(z_0)\)
n = 2
\(2\pi i \cdot f'(z_0)\)
n = 3
\(\pi i \cdot f''(z_0)\)
n = 4
\(\dfrac{\pi i}{3} \cdot f'''(z_0)\)
Use Cauchy when: numerator f(z) is analytic, singularity comes only from (z−z₀)ⁿ in denominator

Which Method to Use?

①
Cauchy's formula Numerator is analytic (eᶻ, sin z, cos z, polynomial). Denominator is (z−z₀)ⁿ only. → differentiate f, plug in z₀
②
Residue theorem f(z) itself has singularities (poles from denominator factors). → find all poles inside C, compute residues, sum × 2πi
③
Cauchy theorem (=0) f(z) is holomorphic everywhere inside C (no singularities at all). → integral = 0

Cauchy–Riemann Conditions

f(z) = u(x,y) + iv(x,y) is holomorphic iff
\(\dfrac{\partial u}{\partial x} = \dfrac{\partial v}{\partial y} \qquad \dfrac{\partial u}{\partial y} = -\dfrac{\partial v}{\partial x}\)
Derivative
\(f'(z) = \dfrac{\partial u}{\partial x} + i\dfrac{\partial v}{\partial x}\)

Laurent Series — Key Technique

Geometric series trick
\(\dfrac{1}{1-w} = \sum_{n=0}^{\infty} w^n, \quad |w| < 1\)
If |z| < |a|: factor out a
\(\dfrac{1}{z-a} = \dfrac{-1}{a}\cdot\dfrac{1}{1-z/a} = \dfrac{-1}{a}\sum_{n=0}^{\infty}\left(\dfrac{z}{a}\right)^n\)
If |z| > |a|: factor out z
\(\dfrac{1}{z-a} = \dfrac{1}{z}\cdot\dfrac{1}{1-a/z} = \dfrac{1}{z}\sum_{n=0}^{\infty}\left(\dfrac{a}{z}\right)^n\)
Always check: is the ratio less than 1? That's the convergence condition.

Taylor Series to Memorize

FunctionExpansion
\(e^z\) \(1 + z + \dfrac{z^2}{2!} + \dfrac{z^3}{3!} + \cdots\)
\(\sin z\) \(z - \dfrac{z^3}{3!} + \dfrac{z^5}{5!} - \cdots\)
\(\cos z\) \(1 - \dfrac{z^2}{2!} + \dfrac{z^4}{4!} - \cdots\)
\(\sinh z\) \(z + \dfrac{z^3}{3!} + \dfrac{z^5}{5!} + \cdots\)
\(\cosh z\) \(1 + \dfrac{z^2}{2!} + \dfrac{z^4}{4!} + \cdots\)

Your Recurring Mistakes — Fix These

❌ Pole order from denominator only
Always expand numerator first. If numerator has zero at same point → order reduces.
e.g. sin(z)/z⁴ → pole order 3, not 4
❌ Sign errors in residues
cos(z) → \(-\sin z\) → \(-\cos z\) → \(\sin z\). Always track the minus sign.
cos(z)/z³ residue is −1/2, not +1/2
❌ Forgetting 1/(n-1)! in order-n formula
For n=3: divide by 2! = 2. For n=4: divide by 3! = 6.
❌ 1/i not simplified
\(\dfrac{1}{i} = -i \qquad \dfrac{1}{2i} = \dfrac{-i}{2} \qquad \dfrac{-1}{i} = i\)
Multiply numerator and denominator by i, remember i² = −1
❌ e^{iz} vs e^z in trig substitution
sin/cos → w = e^{iz}  |  sinh/cosh → w = e^z
❌ Arg confusion
sin(1) ≠ 0.   sin(π) = 0.   sin(1 radian) ≈ 0.841
Don't confuse radian values with multiples of π
