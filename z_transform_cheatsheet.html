Z-Transform Cheatsheet

Everything from the session — methods, rules, traps, and worked patterns
Core Properties
Linearity
Z[af(n) + bg(n)] = a·F(z) + b·G(z)
Always split sums first, then transform each piece separately.
Scaling (most used property)
Z[aⁿ · f(n)] = F(z/a)
Replace every z with z/a in F(z). Then simplify — don't leave z/a in numerator.
Shift — first order
Z[f(n+1)] = z·F(z) − z·f(0)
Shift — second order
Z[f(n+2)] = z²·F(z) − z²·f(0) − z·f(1)
Multiplication by n
Z[n·f(n)] = −z · dF/dz
Summation property
Z[Σf(k)] = z/(z−1) · F(z)
On your formula sheet — you don't need to memorise these, just know how to apply them.
Scaling — Step by Step
Example: Z[2ⁿ · n]
Start with Z[n] = z/(z−1)²
Replace z → z/2:
(z/2) / (z/2 − 1)²
Fix denominator: z/2 − 1 = (z−2)/2
Square it: (z−2)²/4
Divide: (z/2) ÷ ((z−2)²/4)
= (z/2) · (4/(z−2)²) = 2z/(z−2)²
Memorise this pattern instead
Z[aⁿ · n] = az / (z−a)²
a=2 → 2z/(z−2)²  |  a=3 → 3z/(z−3)²
Never leave z/a sitting in the numerator. Always simplify to the clean form.
Going backwards (inverse)
See z/(z−2)² ?
That's (1/2) · 2z/(z−2)², so f(n) = (1/2) · n · 2ⁿ
Rule: squared denominator = n in the answer, always.
Inverse Z-Transform — Partial Fractions
  1. Divide both sides by z: compute F(z)/z
  2. Do partial fractions on F(z)/z
  3. Multiply everything back by z to get F(z) in terms of z/(z−a)
  4. Read f(n) from table: z/(z−a) → aⁿ
Example: F(z) = z / (z−1)(z−3)
F(z)/z = 1/(z−1)(z−3)
= A/(z−1) + B/(z−3)
A: set z=1 → 1/(1−3) = −1/2
B: set z=3 → 1/(3−1) = +1/2
F(z) = −(1/2)·z/(z−1) + (1/2)·z/(z−3)
f(n) = −1/2 + (1/2)·3ⁿ
Repeated root setup — different!
1/(z−a)² → A/(z−a) + B/(z−a)²
Two terms required. Find B first (plug z=a), then find A with any other z value.
Signs — copying A and B into the final answer is where marks get dropped. Double check every sign.
Inverse Z-Transform — Residue Method
Formula
f(n) = Σ Res[ zⁿ⁻¹ · F(z) ] at all poles
The 2πi from the integral formula is already accounted for — do NOT write it in your answer.
Simple pole at z = a
Res = limz→a (z−a) · zⁿ⁻¹ · F(z)
Example: F(z) = (z²−3z)/(z−1)(z−2)
Res at z=1: lim (z−1)·zⁿ⁻¹·F(z)|z=1
= 1ⁿ⁻¹ · (1−3)/(1−2) = (−2)/(−1) = 2
Res at z=2: lim (z−2)·zⁿ⁻¹·F(z)|z=2
= 2ⁿ⁻¹ · (4−6)/1 = 2ⁿ⁻¹·(−2) = −2ⁿ
f(n) = 2 − 2ⁿ
Use residues when partial fractions get ugly. For clean denominators, partial fractions is faster.
Solving Difference Equations — Full Method
Method — always 3 steps
  1. Apply Z-transform to both sides. Use shifting for f(n+1), f(n+2). Z[f(n)] = F(z).
  2. Substitute initial conditions, collect F(z), isolate it algebraically.
  3. Inverse Z-transform via partial fractions or residues → f(n).
Homogeneous — right side = 0
f(n+1) − 2f(n) = 0, f(0)=3
Transform → F(z)(z−2) = 3z
F(z) = 3z/(z−2) → f(n) = 3·2ⁿ
Non-homogeneous — right side ≠ 0
f(n+1) − 2f(n) = 3ⁿ, f(0)=0
Z[3ⁿ] = z/(z−3) goes to the right side
F(z) = z/((z−3)(z−2)) → f(n) = 3ⁿ − 2ⁿ
Second order — hardest type
f(n+2) − 5f(n+1) + 6f(n) = 2ⁿ
f(0)=0, f(1)=1
Z[f(n+2)] = z²F(z) − z²f(0) − z·f(1)
Gives F(z) = z²/((z−2)²(z−3))
Partial fractions: A/(z−2) + B/(z−2)² + C/(z−3)
C=1, B=0 → solve for A → f(n) = A·2ⁿ + 3ⁿ
Always write parentheses: F(z) = 3z/(z−2), never 3z/z−2. Ambiguous notation loses marks even if the algebra is right.
Mistakes Made This Session
❌ Dropping the coefficient in scaling
Z[3ⁿ·n] ≠ z/(z−3)²
Z[3ⁿ·n] = 3z/(z−3)²
The a out front always survives. Check: az/(z−a)².
❌ Confusing scaling with direct table lookup
Z[3ⁿ] is direct from table → z/(z−3)
Z[3ⁿ·n] needs scaling property
Only use scaling when aⁿ is multiplied by another f(n).
❌ Leaving 2πi in residue answer
f(n) = 2πi·(Res1) + 2πi·(Res2) ← wrong
f(n) = Res1 + Res2 ← correct
❌ Sign errors in partial fractions
After finding A and B, copy the exact sign into the final f(n). Most marks lost here.
Quick Pattern Recognition
What you see → what it means
z/(z−a) → aⁿ
az/(z−a)² → n·aⁿ
z/(z−a)² → (1/a)·n·aⁿ
squared denom → n always appears
Factoring denominators — don't mess this up
z²−3z+2 = (z−1)(z−2)
z²+3z+2 = (z+1)(z+2)
z²−5z+6 = (z−2)(z−3)
Always factor before partial fractions. Wrong factoring = everything wrong after.
When you see a rational F(z) with distinct roots → partial fractions. With a repeated root → repeated root setup. With complex algebra → residues.
